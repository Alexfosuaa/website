Intro

I am a sophomore at Yale University, pursuing a Bachelor of Science in Computer Science and Electrical Engineering. My passion for technology and foundation in programming, data structures, and algorithms have fueled my academic journey. I possess a diverse skill set, including proficiency in Java, Python, and web development. My interests primarily focus on software development and machine learning, areas where I am eager to apply and expand my knowledge.

You can explore some of my projects here.

Projects

  • Word Cloud Generator: Developed a Java application that generates a visual representation of the most frequently used words in a given document. Enhanced the project to include interactive flashcards for learning vocabulary.

  • Critter World Simulation: Designed and implemented a Java-based simulation of a 2D world where virtual critters interact according to specified behaviors. Utilized inheritance and polymorphism to create complex interactions between different types of critters.

  • NYT Wordle Game(with 3 words at a time): Created a clone of the popular Wordle game using Java, incorporating features such as grid drawing, user input handling, and game state management.
